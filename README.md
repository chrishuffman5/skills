# Skills

A collection of skills for Claude Code, GitHub Copilot, Gemini, and other AI coding assistants. Each skill lives on its own branch and can be installed independently.

## Installation

From inside your `.claude/skills/` directory, clone the skill branch you want:

```bash
git clone https://github.com/chrishuffman5/skills --branch <skill-name> <skill-name>
```
