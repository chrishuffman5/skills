# Webhooks

### 5.1 `put-webhook`

Defines a webhook and returns a unique webhook URL generated by CodePipeline. The URL can then be registered with a third-party system (GitHub, Bitbucket).

**Synopsis:**
```bash
aws codepipeline put-webhook \
    --webhook <value> \
    [--tags <value>] \
    [--cli-input-json | --cli-input-yaml] \
    [--generate-cli-skeleton <value>]
```

**Parameters:**

| Parameter | Required | Type | Default | Description |
|-----------|----------|------|---------|-------------|
| `--webhook` | **Yes** | structure | -- | Webhook definition |
| `--tags` | No | list | None | Tags. Shorthand: `key=string,value=string ...` |

**Webhook Structure:**
```json
{
    "name": "string",
    "targetPipeline": "string",
    "targetAction": "string",
    "filters": [
        {
            "jsonPath": "string",
            "matchEquals": "string"
        }
    ],
    "authentication": "GITHUB_HMAC|IP|UNAUTHENTICATED",
    "authenticationConfiguration": {
        "AllowedIPRange": "string",
        "SecretToken": "string"
    }
}
```

**Output Schema:**
```json
{
    "webhook": {
        "definition": {
            "name": "string",
            "targetPipeline": "string",
            "targetAction": "string",
            "filters": [
                {
                    "jsonPath": "string",
                    "matchEquals": "string"
                }
            ],
            "authentication": "GITHUB_HMAC|IP|UNAUTHENTICATED",
            "authenticationConfiguration": {
                "AllowedIPRange": "string",
                "SecretToken": "string"
            }
        },
        "url": "string",
        "errorMessage": "string",
        "errorCode": "string",
        "lastTriggered": "timestamp",
        "arn": "string",
        "tags": []
    }
}
```

### 5.2 `delete-webhook`

| Parameter | Required | Type | Default | Description |
|-----------|----------|------|---------|-------------|
| `--name` | **Yes** | string | -- | Webhook name |

### 5.4 `list-webhooks`

Gets a listing of all the webhooks in this account. **Paginated operation.**

**Synopsis:**
```bash
aws codepipeline list-webhooks \
    [--starting-token <value>] \
    [--page-size <value>] \
    [--max-items <value>] \
    [--cli-input-json | --cli-input-yaml] \
    [--generate-cli-skeleton <value>]
```

**Parameters:**

| Parameter | Required | Type | Default | Description |
|-----------|----------|------|---------|-------------|
| `--starting-token` | No | string | None | Pagination token |
| `--page-size` | No | integer | None | Items per API call |
| `--max-items` | No | integer | None | Total items to return |

**Output Schema:**
```json
{
    "webhooks": [
        {
            "definition": {
                "name": "string",
                "targetPipeline": "string",
                "targetAction": "string",
                "filters": [],
                "authentication": "GITHUB_HMAC|IP|UNAUTHENTICATED",
                "authenticationConfiguration": { }
            },
            "url": "string",
            "errorMessage": "string",
            "errorCode": "string",
            "lastTriggered": "timestamp",
            "arn": "string",
            "tags": []
        }
    ],
    "NextToken": "string"
}
```
