## Global Options

All commands accept these options:

| Option | Description |
|--------|-------------|
| `--output` | Output format: `json`, `text`, `table`, `yaml`, `yaml-stream` |
| `--query` | JMESPath query to filter output |
| `--region` | AWS region |
| `--profile` | Named credential profile |
| `--no-cli-pager` | Disable pager for output |
| `--cli-input-json` | Read parameters from JSON string |
| `--cli-input-yaml` | Read parameters from YAML string |
| `--generate-cli-skeleton` | Print JSON/YAML input skeleton |

## Common Patterns

### List all things of a specific type
```bash
aws iot list-things --thing-type-name TemperatureSensor \
    --query 'things[].thingName'
```

### Check device connectivity
```bash
aws iot search-index --query-string "connectivity.connected:true" \
    --query 'things[].{Name:thingName,Connected:connectivity.connected}'
```

### Find certificates expiring soon
```bash
aws iot list-certificates --query 'certificates[?status==`ACTIVE`].{Id:certificateId,Expiry:creationDate}'
```

### Get job execution summary
```bash
aws iot describe-job --job-id my-job \
    --query '{Status:job.status,Targets:job.targets,Summary:job.jobProcessDetails}'
```
