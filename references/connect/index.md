## Common Patterns

### List all instances
```bash
aws connect list-instances \
    --query 'InstanceSummaryList[].{Id:Id,Alias:InstanceAlias,Status:InstanceStatus}'
```

### Search for users by name
```bash
aws connect search-users \
    --instance-id INSTANCE_ID \
    --search-criteria '{"StringCondition":{"FieldName":"Username","Value":"agent","ComparisonType":"STARTS_WITH"}}'
```

### Get real-time queue metrics
```bash
aws connect get-current-metric-data \
    --instance-id INSTANCE_ID \
    --filters '{"Queues":["QUEUE_ID"],"Channels":["VOICE"]}' \
    --current-metrics '[{"Name":"AGENTS_AVAILABLE","Unit":"COUNT"},{"Name":"CONTACTS_IN_QUEUE","Unit":"COUNT"}]'
```
