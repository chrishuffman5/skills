## Global Options

All commands accept these options:

| Option | Description |
|--------|-------------|
| `--output` | Output format: `json`, `text`, `table`, `yaml`, `yaml-stream` |
| `--query` | JMESPath query to filter output |
| `--region` | AWS region |
| `--profile` | Named credential profile |
| `--no-cli-pager` | Disable pager for output |
| `--cli-input-json` | Read parameters from JSON string |
| `--cli-input-yaml` | Read parameters from YAML string |
| `--generate-cli-skeleton` | Print JSON/YAML input skeleton |

## Common Patterns

### Check channel state
```bash
aws medialive describe-channel --channel-id 1234567 \
    --query '{Name:Name,State:State,Class:ChannelClass}'
```

### List running channels
```bash
aws medialive list-channels \
    --query 'Channels[?State==`RUNNING`].{Id:Id,Name:Name}'
```

### Wait for channel to start
```bash
aws medialive wait channel-running --channel-id 1234567
```

### List all inputs with their types
```bash
aws medialive list-inputs \
    --query 'Inputs[].{Id:Id,Name:Name,Type:Type,State:State}'
```
