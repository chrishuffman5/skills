## Global Options

All commands accept these options:

| Option | Description |
|--------|-------------|
| `--output` | Output format: `json`, `text`, `table`, `yaml`, `yaml-stream` |
| `--query` | JMESPath query to filter output |
| `--region` | AWS region |
| `--profile` | Named credential profile |
| `--no-cli-pager` | Disable pager for output |
| `--cli-input-json` | Read parameters from JSON string |
| `--cli-input-yaml` | Read parameters from YAML string |
| `--generate-cli-skeleton` | Print JSON/YAML input skeleton |

## Common Patterns

### List all LoRaWAN devices
```bash
aws iotwireless list-wireless-devices --wireless-device-type LoRaWAN \
    --query 'WirelessDeviceList[].{Id:Id,Name:Name,Status:Status,Destination:DestinationName}'
```

### Check gateway connection status
```bash
aws iotwireless get-wireless-gateway-statistics --wireless-gateway-id GATEWAY_ID \
    --query '{ConnectionStatus:ConnectionStatus,LastUplinkReceived:LastUplinkReceivedAt}'
```

### List FUOTA tasks and their status
```bash
aws iotwireless list-fuota-tasks \
    --query 'FuotaTaskList[].{Id:Id,Name:Name,Status:Status}'
```
